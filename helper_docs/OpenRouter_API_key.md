# OpenRouter API setup
- Visit [Open Router/create key](https://openrouter.ai/sign-in?redirect_url=https%3A%2F%2Fopenrouter.ai%2Fsettings%2Fkeys), create an account / login using your account.  
- Generate a new key (give it any name). 
- Copy the key, once generated. 
    > ⚠️ Never expose that API key. 
- Store that inside `.env` file, 
    ```env
    OPENROUTER_API_KEY = (paste your code)
    ```
